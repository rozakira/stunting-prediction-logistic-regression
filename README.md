# Stunting Prediction using Logistic Regression

Machine Learning project for predicting stunting risk in children based on health and growth data.

This project implements Logistic Regression and includes data preprocessing, model training, evaluation, and deployment using a simple Flask web application.

## 🚀 Features
- Data preprocessing (cleaning & transformation)
- Exploratory Data Analysis (EDA)
- Logistic Regression model
- Model evaluation (accuracy, precision, recall)
- Web-based prediction interface (Flask)

## 📊 Dataset
Health and growth dataset including:
- Age
- Birth weight
- Birth length
- Current weight
- Current height
- Exclusive breastfeeding status

## 🧠 Machine Learning Model
- Algorithm: Logistic Regression
- Library: Scikit-learn
- Output: Stunting risk classification (Yes/No)

## 📄 Publication
This project is part of a research paper published in a SINTA 4 indexed journal.
